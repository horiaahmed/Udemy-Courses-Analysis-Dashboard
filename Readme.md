## Project Overview
This project presents an interactive Power BI dashboard that analyzes Udemy course data including subscribers, price, lectures, subject categories, and yearly course trends.

## Dashboard Pages

### 1. Overview Page
- Subscribers by subject
- Courses by subject (Treemap)
- Top courses by number of subscribers
- Courses by year

### 2. Price Analysis Page
- Average price by subject
- Subscribers by price (Scatter plot)
- Average price vs content duration
- Subscribers, lectures, and reviews by payment category (Free vs Paid)

### 3. Course Details (Drill-Through Page)
This page provides detailed information about a specific course selected from other pages.  
It includes:
- Course title  
- Subject  
- Number of subscribers  
- Number of lectures  
- Number of reviews  
- Course price  
- Content duration (minutes)  
- Slicer with Paid and Free Courses

## Tools Used
- Power BI Desktop  
- Data Transformation using Power Query 
- DAX  

## Dataset Columns
- course_title  
- subject  
- num_lectures  
- num_subscribers  
- num_reviews  
- price  
- content_duration_mins  

## Key Insights
- Web Development has the highest subscriber count.
- Paid courses generate more reviews and lectures.
- Average price ranges between $50 and $100.
- Number of new courses peaked around 2016.
